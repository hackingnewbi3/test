웹 해킹 기초

웹 해킹 공부 전 알아야 할 것들

  • 운영체제 (리눅스)
  • 웹 기본 개념 (웹 구동 원리, 프로토콜 등)
  • 웹 관련 언어: HTML, CSS, JavaScript, PHP, SQL
  • Python (웹 해킹 자동화 및 스크립트 제작)
  • 취약점 개념: Injection (Command, SQL), XSS 등

웹 해킹 실습 순서

  • 웹페이지 만들어보기
  • 풀스택 개발 (프론트엔드, 백엔드)
  • 기능 구현: 로그인, 회원가입, 게시판 (CRUD)

취약점 종류 및 보안

  • Injection (SQL, Command)
  • XSS (Cross-Site Scripting)
  • CSRF (Cross-Site Request Forgery)
  • File 취약점 (업로드, 다운로드)
  • SSRF (Server-Side Request Forgery)

리눅스 기본 명령어

ls -l

디렉토리의 내용을 상세한 리스트 형식으로 출력

cd [디렉토리]

디렉토리를 변경

rm [파일/디렉토리]

파일 또는 디렉토리를 삭제

cat [파일]

파일 내용을 출력

touch [파일]

새 파일을 생성하거나 기존 파일의 수정 시간을 변경

cp [원본] [대상]

파일 또는 디렉토리를 복사

mv [원본] [대상]

파일 또는 디렉토리를 이동하거나 이름 변경

mkdir [디렉토리]

새로운 디렉토리 생성

rmdir [디렉토리]

비어있는 디렉토리 삭제

chmod [권한] [파일]

파일 또는 디렉토리의 권한 변경

chown [사용자:그룹] [파일]

파일 또는 디렉토리의 소유권 변경

ps aux

실행 중인 프로세스 목록 출력

kill [PID]

지정된 프로세스를 종료

top

실시간 시스템 프로세스 및 리소스 사용 현황 출력

df -h

디스크 공간 사용량 확인

du -sh [디렉토리]

디렉토리의 크기 확인

grep [문자열] [파일]

파일에서 특정 문자열 검색

find [디렉토리] -name [파일명]

특정 파일을 찾기 위해 검색

tar -cvf [파일명.tar] [디렉토리]

디렉토리를 tar 파일로 압축

tar -xvf [파일명.tar]

tar 압축 파일 해제

zip [파일명.zip] [파일]

파일을 zip 형식으로 압축

unzip [파일명.zip]

zip 압축 파일 해제
